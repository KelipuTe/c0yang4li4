- 0b_0_10_0x.c

直接输入 2进制、8进制、10进制、16进制的数据。用 16进制的数据验证小端字节序。

- address.c

验证符号表中的地址和程序运行时变量和函数的地址的对应关系。

- setuid.c

特权权限的使用示例。getuid()、geteuid()、setuid()。

- symbol.c
- symbol.elf
- symbol_elf_nm.md
- symbol_elf_objdump.md
- symbol_elf_readelf.md

symbol.c 和 symbol.elf 主要是用来研究 ELF 文件内容的。
